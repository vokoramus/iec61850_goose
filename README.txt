Разработано на Python 3.8, работа проверена на ОС Windows 7.
Для работы программы должны быть установлены библиотеки Python: 
    - scapy, pyasn1, configparser, tkinter, struct


После запуска программы (для удобства можно использовать run.bat) нажать кнопку "Открыть" и выбрать файл "Подстанция 1.ini" (папка "settings")

Можно нажать "Запуск всех гусей" (кнопка с тремя гусями в верхнем меню с кнопками) или кнопку с гусем на любой из вкладок.
Результат (генерация сетевых пакетов) при желании можно наблюдать в любом сетевом сниффере, например, WireShark.
